# 오디오 테스트 파일 폴더

이 폴더에 테스트할 오디오 파일들을 넣어주세요.

## 지원 파일 형식
- MP3 (`.mp3`)
- WAV (`.wav`) 
- M4A (`.m4a`)
- FLAC (`.flac`)
- OGG (`.ogg`)
- WMA (`.wma`)

## 사용 방법
1. 이 폴더에 오디오 파일을 복사
2. 프로젝트 루트에서 `python test_api.py` 실행
3. 모든 오디오 파일이 자동으로 테스트됩니다

## 주의사항
- 이 폴더의 파일들은 Git에 커밋되지 않습니다
- 파일명에 특수문자가 있어도 정상 작동합니다
- 파일 크기가 클 경우 처리 시간이 오래 걸릴 수 있습니다
